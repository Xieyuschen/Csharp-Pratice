## 一些服务：
- Wcf的服务库
&emsp;&emsp;服务库创建的时候会默认有一个接口和一个类，类实现了接口。然后我们需要对接口用特性`[ServiceContractAttribute]`修饰，对于操作/方法使用`[OperationContractAttribute]`来修饰。  
&emsp;&emsp;接口包含了**服务协定**的定义，类是其实现。
